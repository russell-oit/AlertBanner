How to use the Alert Banner

How to use

  • alert.css file contains the styles used by the banner
  • banner.html contains the html fragment
  • test.html contans a browser testable version of the code.

You may insert the style elements directly into your existing stylesheet, import the alert.css file, or include the two styles in the header of your html page. Make the banner div element the first content element of your html page after the head.
